## How to use:
### 1. Clean and build ALL packages in the monorepo ( in root dir ): 
> `yarn && yarn clean`
> `yarn bootstrap`
> `yarn compile`

### 2. Propogate any changes made to ANY of the packages ( in root dir ):
> `yarn compile` 


### 3. Start the examples on localhost ( in examples/* dir ): 
> `yarn start`
